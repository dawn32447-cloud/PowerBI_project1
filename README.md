Project: E-Commerce Sales Dashboard With Power BI


🔖Key Features:

Sales Tracking: Monitor total profit, quantity sold, and average order value.

Geographic Analysis: Filter sales data by State and City (e.g., Delhi, Mumbai, Pune, Indore).

Interactive Slicers: Dynamic filtering using dropdown menus for location-specific insights.

Data Transformation: Cleaned and processed raw CSV/Excel data using Power Query.

Visualizations: Includes bar charts, KPI cards, and trend lines for clear data storytelling.

🏗️Tools Used:

Power BI Desktop: For data modeling and visualization. 

Power Query: For data cleaning and transformation (ETL).

DAX (Data Analysis Expressions): For creating calculated measures and columns.

Dataset: E-commerce sales data (Orders and Details tables).

This E-commerce Sales Dashboard provides a comprehensive, at-a-glance view of business performance, transforming raw data into actionable intelligence.
